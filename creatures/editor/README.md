# Creature Editor Foundation V1

Diese Dateien gehören in:

res://creatures/editor/

Starte zum Testen die Szene:

res://creatures/editor/creature_editor.tscn

Bedienung:
- Linke Leiste: Kategorie und Teil auswählen.
- Rechte Leiste: Stats, Complexity und Transform-Buttons.
- Unten: Name, Save, Load, New, Randomize.
- Rechtsklick ziehen: Kamera drehen.
- Mausrad: Kamera-Zoom.
- Tab: Kategorie wechseln.
- Q/E: aktuelles Bibliotheksteil wechseln.
- Pfeiltasten / PageUp / PageDown: Körperform oder Teilposition ändern.
- Komma/Punkt: kleiner/größer.
- R: Teil rotieren.
- D: Teil duplizieren.
- Entf: Teil löschen.
- M: Spiegelung umschalten.
- Ctrl+S: speichern.
- Ctrl+L: laden.
- Ctrl+R: neue Kreatur.
